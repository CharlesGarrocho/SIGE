# Introdução
O _SIGE_ é um open-source desenvolvido em Java.

Ele foi desenvolvido para um sistema de controle de votação. O _SIGE_ é um trabalho prático da matéria de TOO (Tecnologia em Orientação a Objetos) JAVA.
